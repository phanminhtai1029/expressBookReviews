Implementation Patterns:
------------------------

Pattern 1: Async-Await (Tasks 10 & 12)
---------------------------------------
- Uses async function declaration
- Wraps operations in Promise
- Uses await to wait for Promise resolution
- Handles errors with try-catch blocks
- More readable and maintainable code
- Example:
  async function (req, res) {
    try {
      const result = await somePromise;
      res.send(result);
    } catch (error) {
      res.status(500).json({message: error.message});
    }
  }

Pattern 2: Promise Callbacks (Tasks 11 & 13)
---------------------------------------------
- Creates Promise with new Promise()
- Uses resolve() for success, reject() for errors
- Chains .then() for success handling
- Uses .catch() for error handling
- Traditional Promise pattern
- Example:
  const promise = new Promise((resolve, reject) => {
    if (success) resolve(data);
    else reject(error);
  });
  promise
    .then(data => res.send(data))
    .catch(error => res.status(404).json({message: error}));
